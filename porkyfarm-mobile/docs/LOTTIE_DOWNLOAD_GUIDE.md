# Guide Téléchargement Animations Lottie - PorkyFarm

## Couleurs de la marque

Avant de télécharger, configure ces couleurs dans l'éditeur LottieFiles:

| Couleur | Hex | Usage |
|---------|-----|-------|
| **Primary (Vert)** | `#10B981` | Loading, Success, éléments principaux |
| **Warning (Orange)** | `#F59E0B` | Alertes, coins, bell, food |
| **Error (Rouge)** | `#EF4444` | Erreurs, heart |
| **Pink** | `#EC4899` | Pig, reproduction |
| **Blue** | `#3B82F6` | Calendar |
| **Gray** | `#9CA3AF` | Empty states neutres |

---

## Comment télécharger

1. Clique sur le lien de l'animation
2. Clique sur **"Download"** → **"Lottie JSON"**
3. **IMPORTANT**: Avant de télécharger, clique sur **"Edit Layer Colors"** pour changer les couleurs
4. Renomme le fichier selon le nom indiqué
5. Place dans le dossier correspondant

## Après téléchargement

Une fois les fichiers téléchargés, dis-moi et je mettrai à jour `lib/theme/animations.ts` pour activer les animations.

**Commande pour vérifier la taille des fichiers:**
```bash
ls -lh assets/animations/**/*.json
```

**Taille totale recommandée:** < 500KB pour toutes les animations
